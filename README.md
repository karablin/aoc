# Advent of Code solvers

How to build/run:

```
cd ./2025/c/

./build.sh

./build/1-1c ../input/<yourinput-1>.txt # day 1 task 1
./build/1-2c ../input/<yourinput-1>.txt # day 1 task 2

./build/2-1c ../input/<yourinput-2>.txt # day 2 task 1
./build/2-2c ../input/<yourinput-2>.txt # day 2 task 2

etc...
```
